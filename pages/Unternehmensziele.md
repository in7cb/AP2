# Unternehmensziele

Unternehmensziele beschreiben einen **zukünftigen Zustand**, den die Unternehmensleitung anstrebt. Sie sind Grundlage für Planung, Steuerung und Kontrolle des Unternehmenserfolgs.

## Zielarten

Unternehmensziele werden in drei Kategorien unterteilt:

| Zielart | Untergruppen / Beispiele |
|---|---|
| **Ökonomische Ziele** (wirtschaftlich) | Leistungsziele: Marktanteil erhöhen, Absatz steigern, Qualitätsführerschaft |
| | Erfolgsziele: Gewinnmaximierung, Umsatzsteigerung, Rentabilität |
| | Finanzziele: Liquidität sichern, Eigenkapital sichern, Skontoausnutzung |
| **Soziale Ziele** | Arbeitsplätze sichern, gerechte Entlohnung, Mitarbeiterzufriedenheit, Aufstiegschancen, Mitspracherecht |
| **Ökologische Ziele** | Einhaltung Umweltschutzgesetze, Ressourcenschonung, Abfallvermeidung/-verwertung, CO₂-Reduktion |

> [!tip] **Merksatz**
> **Ö**konomisch – **S**ozial – **Ö**kologisch → **ÖSÖ**: Geld, Mensch, Natur.

## Zielhierarchie

Ziele lassen sich in einer Hierarchie anordnen:

```
Oberziel      → Gewinnmaximierung
  Zwischenziel  → Kostensenkung
    Unterziel     → Produktivität im Fertigungsbereich erhöhen
    Unterziel     → Ausschuss reduzieren

Oberziel      → Gewinnmaximierung
  Zwischenziel  → Absatz erhöhen
    Unterziel     → Produktqualität verbessern
    Unterziel     → Werbemaßnahmen intensivieren
    Unterziel     → Mitarbeiter im Verkauf aufstocken
```

## Zielbeziehungen

Ziele stehen immer in einer Beziehung zueinander:

| Beziehung | Bedeutung | Beispiel |
|---|---|---|
| **Zielkonflikt** (konkurrierend) | Verfolgung von Ziel A beeinträchtigt Ziel B | Umweltschutzmaßnahmen ↔ Gewinnmaximierung; gerechte Entlohnung ↔ Kostensenkung |
| **Zielharmonie** (komplementär) | Förderung von Ziel A begünstigt Ziel B | Hoher Umsatz ↔ hoher Gewinn; Kostensenkung ↔ Rentabilitätssteigerung |
| **Zielneutralität** (indifferent) | Ziel A hat keinen Einfluss auf Ziel B | Ausweitung Vertriebsnetz ↔ Einhaltung Umweltschutzgesetze |

> [!warning] **Achtung Falle**
> Die häufigsten Zielkonflikte in der Prüfung: Gewinnmaximierung vs. Umweltschutz, Gewinnmaximierung vs. Sozialziele (Personalkosten steigen → Gewinn sinkt).

## Zieloperationalisierung

Damit Ziele kontrollierbar sind, müssen sie konkret formuliert werden (Soll-Ist-Vergleich):

| Kriterium | Bedeutung | Beispiel |
|---|---|---|
| **Ausmaß** | Wie viel? | Marktanteil um 5 % steigern |
| **Zeitbezug** | Bis wann? | Innerhalb der nächsten 3 Jahre |
| **Messgröße** | Woran messen? | Marktanteil in % |

Gut formuliertes Ziel: *„Die Reklamationsquote wird im nächsten Jahr auf unter 3 % gesenkt."*

## Betriebliche Kennzahlen

### Produktivität

Misst das mengenmäßige Verhältnis von Output zu Input – **ohne Geldbetrachtung**.

```
Produktivität = Output / Input = erzeugte Menge / eingesetzte Mittel

Arbeitsproduktivität = erzeugte Menge / Anzahl Vollzeitstellen
                    = erzeugte Menge / Anzahl Arbeitsstunden
```

Beispiel: 250.000 Maschinen / 5.000 Vollzeitstellen = 50 Maschinen pro Stelle

### Wirtschaftlichkeit

Misst das **wertmäßige** Verhältnis von Ertrag zu Aufwand.

```
Wirtschaftlichkeit = Umsatz / Kosten

> 1  → wirtschaftlich (Ertrag übersteigt Kosten)
= 1  → kostendeckend
< 1  → unwirtschaftlich (Verlust)
```

> [!important] **Kernregel**
> Produktivität = Menge/Menge (z.B. Stück/Stunde). Wirtschaftlichkeit = €/€ (Umsatz/Kosten). Rentabilität = Gewinn/Kapital × 100 %.

### Rentabilität

Gibt an, wie sich das eingesetzte Kapital **verzinst** hat – relevant für Investoren.

```
Eigenkapitalrentabilität   = Gewinn / Eigenkapital × 100 %

Fremdkapitalrentabilität   = Fremdkapitalzinsen / Fremdkapital × 100 %

Gesamtkapitalrentabilität  = (Gewinn + Fremdkapitalzinsen) / (EK + FK) × 100 %

Umsatzrentabilität         = Gewinn / Umsatz × 100 %
```

| Kennzahl | Aussage | Interessant für |
|---|---|---|
| **EK-Rentabilität** | Verzinsung des Eigenkapitals | Gesellschafter, Aktionäre |
| **GK-Rentabilität** | Gesamte Kapitalverzinsung | Vergleich mit Marktzinsen |
| **Umsatzrentabilität** | Gewinnanteil am Umsatz | Management, Controlling |

### Liquidität

Fähigkeit, jederzeit fällige Zahlungen leisten zu können. **Liquidität geht vor Rentabilität** – ein illiquides Unternehmen ist insolvent, auch wenn es profitabel wäre.

> [!warning] **Achtung Falle**
> Ein Unternehmen kann rentabel aber illiquide sein (z.B. lange Zahlungsziele der Kunden). Insolvenz droht trotz hoher Gewinne auf dem Papier.
