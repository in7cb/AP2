Die Bilanz ist eine Gegenüberstellung von **Vermögen (Aktiva)** und **Kapital (Passiva)** zu einem Stichtag. Bilanzkennzahlen ermöglichen schnelle Aussagen über Finanzierung, Vermögensaufbau, Anlagendeckung und Zahlungsfähigkeit.

## Bilanzstruktur (Strukturbilanz)

Für die Kennzahlenberechnung wird die Bilanz auf eine **Strukturbilanz** verdichtet:

| **Aktiva** | | **Passiva** | |
|---|---|---|---|
| A. Anlagevermögen (AV) | z. B. 700 | A. Eigenkapital (EK) | z. B. 450 |
| B. Umlaufvermögen (UV) | z. B. 850 | B. Fremdkapital (FK) | z. B. 1.100 |
| → Vorräte | 320 | → langfristiges FK | 620 |
| → Forderungen | 280 | → kurzfristiges FK | 480 |
| → flüssige Mittel | 250 | | |
| **Gesamtvermögen** | **1.550** | **Gesamtkapital** | **1.550** |

> [!important] **Kernregel**
> Aktiva = Mittelverwendung (Worin ist Geld gebunden?). Passiva = Mittelherkunft (Woher kommt das Geld?). Bilanz ist immer ausgeglichen: Gesamtvermögen = Gesamtkapital.

## Kennzahlen der Kapitalstruktur

Untersucht die **Finanzierung** (Mittelherkunft): Wie viel EK vs. FK?

```
Eigenkapitalquote = EK / Gesamtkapital × 100

Fremdkapitalquote = FK / Gesamtkapital × 100
```

| Kennzahl | Richtwert | Aussage |
|---|---|---|
| **Eigenkapitalquote** | > 30 % | Hoher EK-Anteil = finanzielle Stabilität, unkündbar, kreditwürdig |
| **Fremdkapitalquote** | < 70 % | Hoher FK-Anteil = höheres Insolvenzrisiko, Zinsbelastung |

Beispiel: EK = 450, GK = 1.550 → EKQ = 450 / 1.550 × 100 = **29 %** (knapp unter Richtwert)

## Kennzahlen der Vermögensstruktur

Untersucht die **Investitionsstruktur** (Mittelverwendung): Wie viel AV vs. UV?

```
Anlageintensität  = Anlagevermögen / Gesamtvermögen × 100

Umlaufintensität  = Umlaufvermögen / Gesamtvermögen × 100
```

| Kennzahl | Aussage |
|---|---|
| **Anlageintensität** | Hoher Wert = kapitalintensiv, wenig Flexibilität (Maschinen, Gebäude gebunden) |
| **Umlaufintensität** | Hoher Wert = flexibel reagierbar, kurzfristig anpassbar |

> [!tip] **Merksatz**
> Anlageintensität + Umlaufintensität = 100 %. Leasing oder Sale-and-lease-back senkt die Anlageintensität.

## Anlagedeckungsgrade (Goldene Bilanzregel)

Zeigt, ob **langfristig gebundenes Vermögen mit langfristigem Kapital** finanziert ist.

```
Anlagedeckungsgrad I  = EK / AV × 100

Anlagedeckungsgrad II = (EK + langfr. FK) / AV × 100
```

| Kennzahl | Richtwert | Bedeutung |
|---|---|---|
| **AD I** | wünschenswert ≥ 100 % | AV vollständig durch EK gedeckt (Idealfall) |
| **AD II** | muss ≥ 100 % | Goldene Bilanzregel – AV mindestens durch EK + langfr. FK gedeckt |

Beispiel: EK = 450, langfr. FK = 620, AV = 700
```
AD I  = 450 / 700 × 100 = 64 %   → AV nicht durch EK allein gedeckt
AD II = (450 + 620) / 700 × 100 = 153 %  → Goldene Bilanzregel erfüllt ✅
```

> [!warning] **Achtung Falle**
> AD II < 100 % bedeutet: Ein Teil des Anlagevermögens wird mit kurzfristigem FK finanziert – Fristenkongruenz verletzt, Insolvenzgefahr.

## Liquiditätsgrade

Misst die **Zahlungsfähigkeit**: Kann das Unternehmen seine kurzfristigen Verbindlichkeiten bedienen?

```
Liquidität 1. Grades = flüssige Mittel / kurzfr. FK × 100

Liquidität 2. Grades = (flüssige Mittel + Forderungen) / kurzfr. FK × 100

Liquidität 3. Grades = (flüssige Mittel + Forderungen + Vorräte) / kurzfr. FK × 100
```

| Grad | Kurzname | Richtwert | Enthält |
|---|---|---|---|
| **L1** | Cash Ratio | 10–30 % | Nur Kasse/Bank |
| **L2** | Quick Ratio | > 100 % | + kurzfristige Forderungen |
| **L3** | Current Ratio | > 100–150 % | + Vorräte (gesamt UV) |

Beispiel: flüssige Mittel = 250, Forderungen = 280, Vorräte = 320, kurzfr. FK = 480
```
L1 = 250 / 480 × 100 = 52 %
L2 = (250 + 280) / 480 × 100 = 110 %  ✅ > 100 %
L3 = (250 + 280 + 320) / 480 × 100 = 177 %
```

> [!warning] **Achtung Falle**
> L2 < 100 % ist kritisch: selbst wenn alle Forderungen eingehen, reicht es nicht zur Begleichung der kurzfristigen Verbindlichkeiten. Zu hohe Liquidität ist aber auch schlecht – ungenutztes Kapital senkt die Rentabilität.

## Leverage-Effekt

Frage: Lohnt sich die Aufnahme von Fremdkapital?

```
Leverage-Effekt:  FK-Aufnahme ist sinnvoll, wenn
                  GK-Rendite > FK-Zinssatz

→ EK-Rendite steigt durch günstige FK-Finanzierung
→ Umgekehrt: GK-Rendite < FK-Zinssatz → EK-Rendite sinkt (Verlustmultiplikator)
```

Beispiel aus Prüfungsaufgabe: EK = 3.200.000 €, FK = 2.500.000 €, GK-Rendite = 11 %, FK-Zinssatz = 8 %
→ **GK-Rendite (11 %) > FK-Zinssatz (8 %)** → Darlehen ist wirtschaftlich sinnvoll ✅

> [!important] **Kernregel**
> Leverage = Hebelwirkung. FK „hebelt" die EK-Rendite hoch – aber nur solange GK-Rendite > FK-Zinsen. Fällt die GK-Rendite darunter, dreht sich der Hebel um und vernichtet EK-Rendite.

## Kreditsicherheiten

Mögliche Sicherheiten bei Bankdarlehen (relevant für Prüfungsaufgaben):

| Dingliche Sicherheiten | Persönliche Sicherheiten |
|---|---|
| Grundschuld / Hypothek (Immobilien) | Bürgschaft |
| Sicherungsübereignung (Maschinen, Fahrzeuge) | Garantie |
| Pfandrecht (bewegliche Sachen) | Schuldbeitritt |
| Forderungsabtretung (Zession) | – |
